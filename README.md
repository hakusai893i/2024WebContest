# 2024WebContest
2024年に作成したWebコンのサイト<br>
サイトに掲載する内容<br>
-各言語の特徴<br>
-変数についての説明（変数自体の宣言、型の宣言など）<br>
-テキストを入出力するための関数についての説明（printfやscanfに類する関数）<br>
-四則演算<br>
-if文に類するコードの説明<br>
-ループ（for, while文）に類するコードの説明<br>
-思想の強すぎない言語を学んだ感想<br>
自作関数を入れるかどうかはちょっと悩んでます<br>
あとこれを学習する際に使ったサイト、書籍などは記録しておいてください<br>
サイトではリンクを日付とともに記録しておいてください　例：https://www/example.com サイトのタイトル名 YYYY/MM/DD<br>
書籍は出版社、著者、ページ数なども記録しておいてください<br>
